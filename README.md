# SETUP

1. Zainstalowane webots. Na Windowsie i Ubuntu instrukcja instalacji w oficjalnej dokumentacji: https://cyberbotics.com/doc/guide/installation-procedure#installation-procedure
1. Po instalacji `git clone` tego repozytorium.
1. W zainstalowanym webots na pasku górnym w `File -> Open World` i otwiewramy plik  `labirynt.wbt`
1. W panelu po prawej `Open an existing text file` z ikoną folderu i otwieramy plik `test_controller.cpp`
1. Klikamy `F7` lub na pasku górnym `Build -> Build`
